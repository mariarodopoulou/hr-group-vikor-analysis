# hr-group-vikor-analysis
HR candidate ranking using Group VIKOR and multi-criteria decision analysis in Python.

## Overview
This project applies the Group VIKOR method to rank HR candidates based on multiple evaluation criteria.

The analysis includes:
- Candidate data preprocessing
- Conversion of categorical HR features into numerical scores
- Group decision-making weights
- VIKOR ranking computation
- Excel export with charts and visualizations

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- XlsxWriter

## Output
The project generates:
- Ranked HR candidate list
- Group VIKOR scores
- Criteria weight charts
- Candidate ranking visualizations
- Excel report with embedded charts
